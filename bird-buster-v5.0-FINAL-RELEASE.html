# 🎮 Bird Buster v5.0 **Juego arcade retro de aviones - Creado por BeastByte** ## 📖 Historia del Proyecto Concepto original: **1986** (Commodore 64 / BASIC) Completado: **2026** (HTML5 + JavaScript con IA Claude) En el año 1986, tuve la idea de crear este juego mientras estudiaba en la universidad. Hice mi primer storyteller pensando en programarlo en lenguaje Basic usando mi Commodore 64 conectado al TV y usando cassettes como unidad de respaldo. Luego de más de 40 años de aquel entonces, logramos completar el código y mejorarlo usando la IA Claude con un prompt con todos los parámetros. ## 🎮 Jugar en línea 👉 **[JUGAR BIRD BUSTER AHORA](https://beastbyte-prg.github.io/bird-buster/bird-buster-v5.0-FINAL-RELEASE.html)** No requiere descargar nada. ¡Solo abre el link y a jugar! ✈️ ## 🕹️ Controles ### Teclado - **⬆️⬇️⬅️➡️** - Mover avión - **SPACE** - Disparar - **Cualquier tecla** - Cerrar modal historia ### Móvil - **D-pad** (abajo derecha) - Mover avión - **🔥 Fuego** (abajo izquierda) - Disparar ### Botones especiales - **🔊/🔇** (arriba derecha) - Mutear/desmutear sonido - **❌** (arriba derecha) - Boss Key (cierre de emergencia) ## 🎯 Objetivo del juego Vuela tu bi-plano rojo, **destruye pájaros**, **recolecta combustible** y **sobrevive** el mayor tiempo posible. ### Puntuación - 🐦 Pájaro volador: **+100 pts** - 🐦 Pájaro en árbol: **+50 pts** - ⛽ Globo combustible: **+200 pts** - ⏱️ Cada segundo: **+1 pt** ## 🎵 Sistema de Audio 11 sonidos únicos incluidos: - 🎵 Música de título retro arcade - ✈️ Motor WWII sputtering - 💥 Disparos tipo revólver - 💣 Explosiones dinámicas - 🌙 Melodía anochecer (descendente) - ☀️ Melodía amanecer (ascendente) - 🔔 Campanadas de récord - ⛽ Sonido recarga de combustible - Y más... ## 🌟 Características ✅ **Juego completo autocontenido** - Un único archivo HTML, sin dependencias externas ✅ **Compatible con cualquier navegador** - Chrome, Firefox, Safari, Edge ✅ **Funciona offline** - Juega sin internet ✅ **Responsive** - Juega en desktop, tablet o móvil ✅ **Ciclo día/noche** - El cielo cambia suavemente entre día y noche ✅ **Sistema de combustible** - Estrategia: ¿volar alto (consume más) o bajo? ✅ **Dificultad progresiva** - Cada 1000 puntos sube la velocidad ✅ **Highscores** - Se guardan en tu navegador (localStorage) ✅ **Sonido envolvente** - Web Audio API con efectos dinámicos ✅ **Boss Key** - Ciérrate rápido si viene el jefe 😅 ## 💾 Versiones | Versión | Cambio | |---------|--------| | v1.0 | Concepto básico | | v2.x | Día/noche + sonidos | | v3.0 | Orientación adaptativa | | v4.0-4.6 | Mecánicas avanzadas + efectos | | **v5.0** | **FINAL RELEASE** - Boss Key, Mute, Modal mejorado | ## 👨‍💻 Créditos **Creador:** José R.
